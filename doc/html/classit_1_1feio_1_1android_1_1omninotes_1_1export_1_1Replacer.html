Detailed Description

Helper class for the HTML exporter. The purpose of the class to go through an array of strings and replace all strings starting with $ with a information from the note. The caller registers a number ReplaceFunc implementations, one for each specific $-strings. ReplaceFunc.replace is called when the associated $-string is found, the returned text is sued to replace the $-string.

Detailed Description

An note exporter has to implement this interface. This is used by the rest of the application to access the exporter functionality.

OutputStream  os

Takes a note and exports all of it's information to an OutputStream in a format specific to the implementing exporter. Which information is exported depends on the resulting file format of the exporter, for example pictures might not be exported to a text based format.
